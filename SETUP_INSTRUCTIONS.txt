IMAGE LINKING
═══════════════════════════════════════════════════════════════════════════════

How it works:
  1. Product name is slugified: "River 2 Max" → "river-2-max"
  2. System looks for: /assets/images/products/river-2-max.png
  3. If found: Uses that image
  4. If not found: Uses fallback /assets/images/products/river-2-max.svg
  5. Frontend renders with proper fallback initials if no image

Required folder structure:
  /assets/images/products/
    ├── river-2-max.png
    ├── river-2-pro.png
    ├── delta-2-max.png
    └── ... (other product images)
